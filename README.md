# ideaSQL-instagram-crawler
## 用法

在mac上安裝 firefox
安裝python selenium

-q : hashtag   /  -n 數量

## 執行抓取
```
python instagramcrawler.py -q "#端午節" -n 3000
python instagramcrawler.py -q "#台中" -n 3000
```

## 上傳抓取檔案
旁邊會自動建立名稱為：result_#台中小吃_2000.txt 類似的檔名
到上傳系統 http://designav.io/image
清空資料庫後，選擇群組加入
選取文字檔案上傳即可
